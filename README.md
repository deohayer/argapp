# Installation

## Prerequisites

### OS

 * Linux, tested with:
    * Ubuntu 18.04
    * Ubuntu 19.10
    * Ubuntu 20.04
    * Ubuntu 21.10
    * Ubuntu 22.04

### Python versions

 * 3.6
 * 3.7
 * 3.8
 * 3.9
 * 3.10
 * 3.11

## Steps

To install, run the following:
```bash
pip3 install argapp
```

To enable the completion, install [argcomplete](https://pypi.org/project/argcomplete).<br>
Refer to its documentation for the details on installation and usage.
